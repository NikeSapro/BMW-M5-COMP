# BMW M5 Competition - Moderní webová prezentace

Kompletní, plně optimalizovaná webová prezentace zaměřená na specifikace, design a zajímavosti o ikonickém supersedanu BMW M5 Competition. Projekt slouží jako ročníková práce pro předmět Webové technologie.

* **Odkaz na živý web:**  https://nikesapro.github.io/rocnikova-prace-levandovsky/


## Prohlášení o vypracování projektu
Tento projekt byl vypracován autorem samostatně na základě reálných automobilových specifikací a grafických podkladů vozu BMW M5. Během kódování a optimalizace responzivity byla jako pomocný a optimalizační nástroj využita umělá inteligence (AI), která asistovala při odstraňování chyb v kódu a ladění zobrazení pro mobilní telefony. Finální podoba webu je dílem autora.


## 1. Použité technologie

* **HTML5:** Sémantické tagy (header, nav, section, footer) pro logickou a přehlednou strukturu obsahu.
* **CSS3:** Moderní responzivní layouty (Flexbox pro navigaci, CSS Grid pro galerii) a Media Queries pro bezchybné zobrazení na mobilu i PC.
* **Vanilla JavaScript (ES6+):** Programování klientské logiky bez externích knihoven, ošetření odeslání kontaktního formuláře a výpis alertu.
* **Vývojové prostředí:** Operační systém Windows 11, textový editor / IDE Visual Studio Code.
